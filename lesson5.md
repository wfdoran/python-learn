## Boolean Variables

```python
x = 5 > 3
y = 5 > 8
z = 5 == 5
print(x,y,z)
print(type(x))
```

A boolean variable takes on the value `True` or `False` and 
indicates whether an expression is correct.  The basic 
comparision operations are

| operator | description | examples     |
| ---------| ----------- | ---------    |
| `==` | equal     | `5 == 5` => `True`  |
|      |           | `6 == 5` => `False` |
| `!=` | not equal | `6 != 5` => `True`  |
|      |           | `5 != 5` => `False` |
